Overview

The CRIKIT webtool is a Shiny web application for the analysis and visualization of well plate fluorescence data, designed for CRISPR-Cas13-based antibiotic resistance detection. The app supports multiple data formats, interactive visualizations, and modular analysis workflows.

Getting Started

Prerequisites

Before you begin, ensure you have the following installed:

  • R version 4.0 or higher
  • R packages: shiny, ggplot2, DT, bslib, tidyr, reshape2, readxl, readr, tools, stringr

Installation

  1. Clone the repository

    git clone https://gitlab.igem.org/2025/software-tools/groningen
cd groningen/RShiny

  2. Install R and required packages

    install.packages(c("shiny", "ggplot2", "DT", "bslib", "tidyr", "reshape2", "readxl", "readr", "tools", "stringr"))

Usage

From the RShiny directory, launch the app with: 

Rscript app_runner.r

Or, in an R console: 

setwd("path/to/RShiny")
source("app_runner.r")

The app will open in your default web browser. If not, copy the URL from the terminal and open it manually.

You can upload your datasets via the app interface. Supported formats: CSV, TSV, TXT, Excel (Tecan SparkControl).

Data Format

Delimited files (CSV, TSV, TXT) must contain columns:

  • Cycle Nr.: PCR cycle (required)
  • Time: Time point (optional)
  • A1-H12: Well values for a 96-well plate
